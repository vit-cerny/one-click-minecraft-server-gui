How to run

Put both files in the same folder (e.g. C:\MCManager\)
Double-click START.bat

Requires Python 3.8+ from python.org — check "Add Python to PATH" during install.

What's included
Create Server tab — three sub-tabs:

Paper — fetches live version list from PaperMC API, downloads latest build automatically
Fabric — fetches live Fabric loader + installer versions, creates /mods folder
Vanilla — fetches from Mojang's official manifest

Console — real-time log output, command input, clear button
Mods (Fabric) — search Modrinth live, see downloads/description, one-click install into /mods, scan/remove installed mods
Plugins (Paper) — 16 curated recommended plugins with direct download links, scan/remove installed plugins
server.properties — quick-settings dropdowns for gamemode/difficulty/pvp/port etc, plus full raw editor with save
playit.gg Tunnel — one-click download of the playit agent, start/stop tunnel, live tunnel output showing your public address
Settings — Java path auto-detect or browse, default RAM, link to Java 21 download
